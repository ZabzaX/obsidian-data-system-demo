# Obsidian Data System Demo
This project demonstrates a structured data system built in Obsidian using YAML metadata, Dataview, and DataviewJS.

## What this shows
This project demonstrates how structured data (YAML) can be used to design interconnected game systems (quests, locations, NPCs) inside Obsidian.

## Features
- Relational data modeling (regions, NPCs, locations, events)
- Dynamic queries and dashboards
- Custom scripting (DataviewJS – shop system)
- Quest and level design integration
